# DS3000 Bridge Infrastructure

## About
This repository contains code and resources for analyzing and modeling civil infrastructure datasets, specifically focusing on bridge data from various US states.

## Data Files
Some of the dataset files in this project are very large and **cannot be committed directly to GitHub** due to file size restrictions:

- **GitHub file size limit:** 100 MB per file
- **Git LFS limit:** Up to 2 GB per file (with bandwidth/storage quotas)
- **Some files in this project exceed these limits**

### How to Access Large Datasets
- **Files larger than 100 MB** are not included in this repository.
- If you need access to the full datasets, please contact the repository owner or use the provided cloud storage links (if available).
- Only small sample datasets or code to download/process the data are included here.

#### Recommended Workflow
1. **Do not commit files over 100 MB** to the repository.
2. For files between 100 MB and 2 GB, use [Git Large File Storage (LFS)](https://git-lfs.github.com/).
3. For files over 2 GB, upload them to a cloud storage service (Google Drive, Dropbox, OneDrive, etc.) and share the download link here.

#### Example Table
| File Size         | Can Commit Directly? | Use Git LFS? | Use Cloud Storage? |
|-------------------|---------------------|--------------|--------------------|
| < 100 MB          | Yes                 | Optional     | Optional           |
| 100 MB – 2 GB     | No                  | Yes          | Optional           |
| > 2 GB            | No                  | No           | Yes                |
